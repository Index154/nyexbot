WHEN TO PUT STUFF WHERE
- SQL: Data that is often altered at runtime and data with special filtering requirements
- JS files (const variables): Data which is never altered at runtime and should always remain in memory due to frequent usage
- TXT files: Data which is occasionally altered at runtime. Simple data that is easy to read and modify manually
- JSON files: Data which is occasionally altered at runtime. Complex or relational data that is otherwise hard to read and modify manually
